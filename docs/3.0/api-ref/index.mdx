Syntask auto-generates reference documentation for the following components:

- **[Syntask Python SDK](https://syntask-python-sdk-docs.netlify.app/)**: used to build, test, and execute workflows.
- **[Syntask REST API](/3.0/api-ref/rest-api)**: used by workflow clients and the Syntask UI for orchestration and data retrieval.
  - Syntask Cloud REST API documentation: <a href="https://app.syntask.cloud/api/docs" target="_blank">https://app.syntask.cloud/api/docs</a>.
  - Self-hosted Syntask server [REST API documentation](/3.0/api-ref/rest-api/server/). Additionally, if self-hosting a Syntask server instance, you can access REST API documentation at the `/docs` endpoint of your [`SYNTASK_API_URL`](/3.0/manage/settings-and-profiles/). For example, if you run `syntask server start` with no additional configuration you can find this reference at <a href="http://localhost:4200/docs" target="_blank">http://localhost:4200/docs</a>.
